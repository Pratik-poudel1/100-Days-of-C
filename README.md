# 100-Days-of-C
Daily C programming challenge, 100 days. 5 to 10 problems per day.
100 Days of C Programming

This repository contains my structured 100 day problem solving plan in C.

I follow a fixed schedule.
I solve 5 to 10 problems each day.
I progress from basic concepts to advanced algorithms.

The plan includes more than 500 problems across 100 days.

Topics covered:

• Basic syntax, input output, operators
• Decision making and loops
• Functions and recursion
• Arrays and matrices
• Strings and pointers
• Dynamic memory allocation
• File handling
• Structures, unions, enums
• Linked lists
• Stack and queue
• Trees and graphs
• Hashing and heaps
• Dynamic programming
• Backtracking
• System programming

Each section builds on previous concepts.
I strengthen my logic through daily coding practice.
I write complete C programs for every problem.

This repository documents my C practice roadmap for exam preparation, internships, and technical interviews.
